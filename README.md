# LumosCareer

LumosCareer is a web app for career guidance, interview preparation, and mental well-being, customized to help candidates navigate their professional journeys effectively.

## Core Features

- **Compatibility Reports:** Generates compatibility reports between a candidate’s profile and a company’s background.
- **Career Path Generation:** Recommends dynamic, step-by-step career roadmaps based on a user's resume, bio, and targets.
- **AI Assessments:** Evaluates user traits (MBTI-style) to help tailor career paths and company matching.
- **Company Scraping & Analysis:** Provides deep dive insights into saved companies using AI and webhooks.

## Integrated Voice Agents

- **[Dr. Hannibal Lecter - AI Therapist](https://hannibal-by-bhaveshpatil75.vercel.app)**: A refined, calm therapist persona offering insightful therapy sessions for stress relief.

## Technologies Used

- **Backend:** Python, Django, SQLite
- **Frontend:** HTML, CSS, JavaScript (via Django Templates)
- **AI & Real-Time Sync:** Gemini API, LiveKit Core
- **Automation:** n8n Webhooks (for scraping and data paths)
- **Core Algorithms:** PageRank, Dijkstra, Simulated Annealing (for Career Paths and Market Signals)
